# Token Rules

All color, spacing, radius, shadow values MUST use CSS custom property tokens. NEVER hardcode hex, rgb, oklch, or Tailwind color primitives (gray-100, zinc-800, etc.).

## Color Tokens

**Surfaces:** `bg-background` · `bg-card` · `bg-popover` · `bg-muted` · `bg-accent`
**Text:** `text-foreground` · `text-card-foreground` · `text-muted-foreground` · `text-primary-foreground`
**Actions:** `bg-primary` · `bg-secondary` · `bg-destructive` · `border-border` · `ring-ring`
**Chart:** `var(--chart-1)` ~ `var(--chart-5)` (generated by `npx shadcn init`)
**Dashboard extensions** (`scripts/templates/custom-tokens.css`): `var(--kpi-bg)` · `var(--kpi-positive)` · `var(--kpi-negative)` · `var(--table-row-hover)`

## Radius Tokens

`rounded-[--radius]` · `rounded-[--radius-sm]` · `rounded-[--radius-lg]`
NEVER use `rounded-md`, `rounded-lg` — use token-based form.

## Typography

한국어 기준. 본문 기본 `text-sm`(14px). 한국어 글리프는 16px이 대시보드에서 과도하게 크다.

| 역할 | 클래스 | px | Weight |
|------|--------|-----|--------|
| 본문 (기본) | `text-sm` | 14 | `font-normal` |
| 보조 텍스트, 캡션 | `text-xs` | 12 | `font-normal` |
| 카드 제목 (CardTitle) | `text-base` | 16 | `font-semibold` |
| 페이지 제목 (h1) | `text-xl` | 20 | `font-semibold` |
| KPI 값 | `text-2xl` | 24 | `font-semibold` |
| 라벨 (FieldLabel 등) | `text-sm` | 14 | `font-medium` |

## Spacing

Tailwind 4px 단위 spacing scale만 사용. `gap-4`(16px)가 기본, `gap-2`(8px)가 인라인 기본.

**허용 단위:** `gap-1`~`gap-6` · `p-1`~`p-6` · `m-1`~`m-6` · `space-y-*` · `space-x-*`

| 역할 | 클래스 | px |
|------|--------|-----|
| 페이지 루트 래퍼 | `flex flex-col gap-4 p-4` | gap 16, padding 16 |
| 페이지 헤더 | div (Card 아님) | — |
| 섹션 간 간격 (KPI↔Chart↔Table) | 루트의 `gap-4`가 처리 | 16 |
| Card grid (KPI, Chart) | `gap-4` | 16 |
| CardContent 내부 수직 그룹 | `space-y-4` | 16 |
| 필터 바 ↔ DataTable 간격 | `pb-4` (필터 바에 적용) | 16 |
| FieldGroup 내 Field 간격 | FieldGroup 기본 gap | — |
| FieldSet 내 grid (2-col) | `gap-4` | 16 |
| 인라인 요소 간 수평 간격 | `gap-2` | 8 |
| 버튼 그룹 (CardFooter 등) | `gap-2` | 8 |

**금지:** inline style spacing, Tailwind scale 밖 임의 값(`gap-[13px]`), 기본값과 다른 간격을 지시 없이 변경.

## Chart Config

```tsx
const chartConfig = {
  desktop: { label: "Desktop", color: "var(--chart-1)" },
  mobile: { label: "Mobile", color: "var(--chart-2)" },
} satisfies ChartConfig
```

Chart elements reference colors as `var(--color-KEY)` (e.g., `stroke="var(--color-desktop)"`). Charts live inside `Card > CardContent > ChartContainer`.

## Escape Hatch

새 토큰이 필요하면 `scripts/templates/custom-tokens.css`에 먼저 추가한 뒤 `var(--token)`으로 참조. inline literal 사용 후 "나중에 추가" 금지.
