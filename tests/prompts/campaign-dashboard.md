# 캠페인 대시보드

## 배경 및 목적

광고 운영팀은 매일 아침 전날 캠페인 성과를 빠르게 점검하고, 주간 팀 회의에서 전체 현황을 공유해야 한다. 현재는 여러 채널의 리포트를 수작업으로 취합하고 있어 시간 소요와 오류가 발생한다.

이 대시보드는 모든 채널의 주요 성과 지표를 단일 화면에서 확인하고, 이상 징후를 조기에 포착하며, 데이터에 근거한 의사결정을 지원한다.

## 사용자

**주 사용자: 광고 운영 매니저**
- 매일 아침 가장 먼저 열어 전날 성과를 확인한다.
- 이상이 있는 캠페인을 식별하고 상세 화면으로 이동하여 조치한다.

**부 사용자: 광고 전략 팀장**
- 주간 회의에서 프로젝터로 띄워 현황을 공유한다.
- 채널별 예산 배분 현황을 파악하여 전략 의사결정에 활용한다.

## 핵심 지표 (Summary KPIs)

아래 4가지 지표를 표시한다. 각 지표에 직전 동일 기간 대비 변화율을 함께 표시하여 추세를 즉시 파악할 수 있도록 한다.

| 지표 | 현재 값 | 전주 대비 |
|------|---------|----------|
| 총 지출 | 245,800,000원 | +12.5% |
| 노출수 | 18,420,000 | +8.3% |
| 클릭수 | 892,000 | -2.1% |
| 평균 CTR | 4.84% | +0.5% |

변화율은 증가/감소 방향에 따라 시각적으로 구분한다.

## 시각화 요구사항

### 일별 광고비 추세

- **목적:** 광고비 집행 패턴과 급격한 증감 시점을 파악한다.
- **데이터:** 일 단위 광고비 합계, 기본 30일.
- **기간 전환:** 7일 / 30일 / 90일 중 선택 가능. 선택 즉시 갱신.
- 목 데이터 30건

### 채널별 광고비 비중

- **목적:** 전체 예산 중 각 채널이 차지하는 비중을 파악한다.
- **데이터:** Google Ads, Meta Ads, Naver SA, Kakao Moment, X Ads 채널별 광고비 합계.
- 각 채널이 구분되어 비교 가능해야 한다.

## 최근 캠페인 목록

"최근 캠페인" 제목, "최근 활동이 있는 캠페인" 부제.

표시 항목: 캠페인 ID, 캠페인명, 상태(활성/일시정지/종료), 채널, 노출수, 클릭수, CTR, 지출.

- 주요 수치 컬럼은 정렬 가능
- 행 클릭 시 `/campaigns/{id}`로 이동
- 50행 목 데이터 (다양한 상태와 채널 혼합, 페이지네이션 테스트 가능)

## 사용자 흐름

### 아침 점검
1. 대시보드 접속 → KPI에서 전일 대비 변화 확인
2. 이상 지표 발견 → 채널별 비중에서 쏠림 확인
3. 캠페인 목록에서 문제 캠페인 찾아 행 클릭 → 상세 페이지로 이동

### 주간 회의
1. 프로젝터에 대시보드를 띄움
2. 일별 추세로 30일 광고비 흐름 설명
3. 채널별 비중으로 채널 전략 토론

## 기술 제약

- 한국어 UI, 원화 표기 (원 접미사, ₩ 사용 금지)
- 단일 페이지 컴포넌트, 목 데이터 인라인
