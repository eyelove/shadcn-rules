# 캠페인 상세

## 배경 및 목적

캠페인 리스트에서 집계 수치만으로는 원인 분석이 불가능하다. 이 페이지는 개별 캠페인의 성과를 심층 분석하고, 채널별 기여도를 비교하며, 하위 광고그룹 단위까지 드릴다운할 수 있는 캠페인 단위 의사결정 허브다.

## 사용자

| 페르소나 | 주요 관심사 |
|----------|------------|
| 광고 운영 담당자 | 캠페인 성과가 목표 대비 어느 수준인지, 어느 채널이 효율적인지 |
| 광고주 담당자 | 예산 소진 속도와 전환 성과 요약 |

## 핵심 지표 (KPI)

아래 4개 지표를 최상단에 노출한다. 각 지표는 전월 대비 증감률과 "전월 대비" 보조 텍스트를 함께 표시한다.

| 지표 | 현재값 | 전월 대비 |
|------|--------|----------|
| 총 지출 | 42,500,000원 | +15.2% |
| 노출수 | 3,280,000 | +22.1% |
| 클릭수 | 156,000 | -3.4% |
| 전환율 | 3.82% | +0.8% |

증감률은 양수/음수 방향에 따라 시각적으로 구분한다.

## 캠페인 정보

페이지가 나타내는 캠페인의 기본 속성이다. 맥락 파악용이며 수정 기능은 범위 밖이다.

| 항목 | 값 |
|------|----|
| 캠페인명 | 2026 봄 프로모션 |
| 상태 | 활성 |
| 채널 | Google Ads |
| 시작일 | 2026-03-01 |
| 종료일 | 2026-04-30 |
| 일 예산 | 12,500,000원 |
| 총 예산 | 375,000,000원 |
| 광고주 | (주)테크스타 |

## 성과 시각화

### 일별 성과 추이

- **목적:** 최근 기간 동안 집행 금액과 전환 수가 어떻게 변화했는지 파악한다.
- **데이터:** 일별 지출, 일별 전환수 (두 지표 동시 표현)
- **기간 전환:** 7일 / 30일 / 90일 선택 가능, 기본 30일
- 범례(legend)로 지표를 식별할 수 있어야 한다.
- 목 데이터 30건

### 채널별 성과 비교

- **목적:** Google Ads, Meta Ads, Naver SA 세 채널의 기여도를 비교한다.
- **데이터:** 채널별 지출, 클릭수, 전환수
- 세 채널을 동시에 비교할 수 있어야 한다.

## 광고그룹 목록

캠페인에 속한 광고그룹 전체를 조회하고 상태별로 필터링한다.

"광고그룹" 제목, "이 캠페인에 속한 광고그룹" 부제.

### 상태별 탭

| 탭 | 광고그룹 수 |
|----|-----------|
| 활성 (기본) | 25개 |
| 일시정지 | 15개 |
| 종료 | 10개 |

### 테이블 항목

캠페인 ID 대신 광고그룹 ID, 광고그룹명, 상태, 노출수, 클릭수, CTR, 지출, CPA.

- 수치 컬럼은 정렬 가능
- 총 50행 목 데이터 (페이지네이션 테스트 가능)
- 행 클릭 시 `/ad-groups/{id}`로 이동

## 사용자 흐름

1. 캠페인 목록에서 행 클릭으로 진입한다.
2. 페이지 상단에 캠페인 목록으로 돌아가는 수단이 있다.
3. 캠페인명 옆에 상태가 표시된다.
4. KPI → 시각화 → 광고그룹 순으로 분석한다.
5. 광고그룹 행 클릭 시 `/ad-groups/{id}`로 이동한다.
