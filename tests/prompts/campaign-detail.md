# 캠페인 상세

캠페인 운영자가 개별 캠페인의 성과를 분석하는 상세 페이지입니다.
상단에서 핵심 지표를 빠르게 확인하고, 차트로 추이를 분석한 뒤, 하단에서 소속 광고그룹의 세부 성과를 볼 수 있어야 합니다.

## 페이지 헤더
- 캠페인 목록(/campaigns)으로 돌아가는 뒤로가기 버튼
- 제목: "2026 봄 프로모션" (캠페인명)
- 설명: "캠페인 상세 성과 분석"
- 우측에 상태 표시 (활성)

## 핵심 지표 (4개)

| 지표 | 값 | 전월 대비 | 보조 텍스트 |
|------|-----|----------|-----------|
| 총 지출 | 42,500,000원 | +15.2% | 전월 대비 |
| 노출수 | 3,280,000 | +22.1% | 전월 대비 |
| 클릭수 | 156,000 | -3.4% | 전월 대비 |
| 전환율 | 3.82% | +0.8% | 전월 대비 |

- 금액/수량은 만/억 단위로 축약 (예: 4,250만원, 328만)
- 증감은 양수 초록, 음수 빨강
- 보조 텍스트는 작은 글씨

## 캠페인 정보 요약

하나의 영역에 캠페인 메타 정보를 모아 표시합니다.

- 상단: 상태(라벨), 채널(텍스트), 시작일, 종료일 — 2열 배치
- 구분선
- 하단: 일일 예산(12,500,000원), 총 예산(375,000,000원), 광고주명("(주)테크스타")
- 날짜는 YYYY-MM-DD 형식

## 차트 (2개, 나란히 배치)

### 일일 성과 추이
- 최근 30일 선 그래프
- 2개 지표: 지출, 전환수 (각각 다른 색상)
- 기간 변경: 7일 / 30일 / 90일
- 범례(legend) 표시
- 호버 시 툴팁
- 목 데이터 30건

### 매체별 성과 비교
- Google Ads, Meta Ads, Naver SA — 3개 채널
- 각 채널별 지출 / 클릭 / 전환을 묶은 막대 그래프 (grouped bar)
- 각 지표 다른 색상

## 광고그룹 목록 (상태별 탭 전환)

"광고그룹" 제목, "이 캠페인에 속한 광고그룹" 부제

- 탭: "활성" (기본), "일시정지", "종료"
- 각 탭의 테이블 컬럼:
  - ID, 광고그룹명, 상태, 노출수, 클릭수, CTR, 지출, CPA
  - 숫자 컬럼 오른쪽 정렬, 천단위 구분자
  - 금액순 정렬 가능
- 활성 탭 25행, 일시정지 탭 15행, 종료 탭 10행 목 데이터 (총 50행, 페이지네이션 테스트 가능하도록)
- 행 클릭 시 /ad-groups/{id}로 이동

## 기타
- 한국어 UI, 원화 표기
- 지표 카드에는 축약 포맷, 테이블에는 정확한 숫자
- 단일 페이지 컴포넌트, 목 데이터 인라인
