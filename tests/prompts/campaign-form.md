# 캠페인 생성

## 배경 및 목적

광고 관리 플랫폼에서 운영자가 새로운 광고 캠페인을 생성하는 핵심 워크플로우다. 캠페인 목록 페이지에서 진입하여, 기본 정보·운영 목표·예산·일정을 한 번에 정의할 수 있어야 한다.

- 운영자가 실수 없이 올바른 설정으로 캠페인을 등록할 수 있도록 안내한다.
- 필수 정보 누락이나 잘못된 값 입력 상태로 캠페인이 생성되는 것을 방지한다.
- 생성된 캠페인이 즉시 광고 운영 흐름에 편입될 수 있도록 충분한 초기 정보를 수집한다.

## 사용자

| 페르소나 | 설명 |
|---------|------|
| 광고 운영자 | 대행사 소속으로 여러 광고주의 캠페인을 동시에 다루며 빠른 등록을 원한다. |
| 광고주 담당자 | 플랫폼 숙련도가 낮을 수 있으므로 각 항목의 의미와 제약이 명확해야 한다. |

## 요구사항

### 1. 기본 정보

| 필드 | 필수 여부 | 제약 조건 |
|------|---------|---------|
| 캠페인 이름 | 필수 | 최소 2자, 최대 100자 |
| 광고주 | 필수 | 30개 이상 광고주 목록에서 선택. 단일 선택. |
| 캠페인 설명 | 선택 | 최대 500자. 운영자 내부 메모 용도. |

### 2. 캠페인 설정

| 필드 | 필수 여부 | 제약 조건 |
|------|---------|---------|
| 캠페인 목표 | 필수 | 브랜드 인지도, 전환, 트래픽 중 단일 선택. |
| 캠페인 유형 | 필수 | 검색 광고("키워드 기반 검색 결과에 노출"), 디스플레이 광고("배너 및 이미지 형태로 노출"), 동영상 광고("영상 콘텐츠로 노출") 중 단일 선택. 각 유형에는 제목과 설명이 함께 제공되어야 한다. |
| 채널 | 필수 | Google Ads, Meta Ads, Naver SA, Kakao Moment 중 단일 선택. |
| 자동 최적화 | 선택 | 활성화 시 입찰가를 자동 조정. 기본값 비활성. 설명: "성과에 따라 입찰가를 자동으로 조정합니다" |

### 3. 예산 및 일정

| 필드 | 필수 여부 | 제약 조건 |
|------|---------|---------|
| 일일 예산 | 필수 | 원화(KRW) 기준. 최솟값 1,000원. 안내: "일일 최대 지출 금액 (원)" |
| 시작일 | 필수 | 날짜 선택 |
| 종료일 | 필수 | 시작일 이후 날짜 |

시작일과 종료일은 같은 섹션 내에서 나란히 배치한다.

## 유효성 검사

| 검사 항목 | 실패 조건 | 메시지 |
|---------|---------|--------|
| 캠페인 이름 | 미입력 | "캠페인명은 필수입니다" |
| 일일 예산 | 미입력 | "예산은 필수입니다" |
| 일일 예산 | 1,000원 미만 | "최소 1,000원 이상" |

검사 실패 시 해당 필드에 오류 메시지를 표시한다.

## 사용자 흐름

1. 캠페인 목록 페이지에서 "캠페인 생성" 버튼으로 진입한다.
2. 페이지 상단에 캠페인 목록으로 돌아가는 수단이 있다.
3. 기본 정보 → 캠페인 설정 → 예산/일정 순으로 입력한다.
4. "저장" 시 유효성 검사 통과하면 캠페인 목록으로 돌아간다.
5. "취소" 시 입력 내용을 저장하지 않고 캠페인 목록으로 돌아간다.

## 기술 제약

- 한국어 UI
- 단일 페이지 컴포넌트, 목 데이터 인라인
