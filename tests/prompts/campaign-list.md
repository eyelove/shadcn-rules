# 캠페인 관리

## 배경 및 목적

광고 관리 플랫폼에서 운영 담당자가 집행 중인 모든 캠페인을 한 화면에서 파악하고, 상태 변경·편집·삭제·내보내기 등 일상적인 운영 작업을 수행할 수 있어야 한다. 이 페이지는 캠페인 운영의 단일 진입점이다.

## 사용자

| 페르소나 | 주요 관심사 |
|---------|-----------|
| 광고 운영 담당자 | 집행 현황 파악, 빠른 ON/OFF 전환, 하루에 수십 번 방문 |
| 광고 기획자 | 신규 캠페인 등록 및 기존 캠페인 수정 |
| 팀장 / 관리자 | 전체 캠페인 성과 조회 및 보고용 데이터 추출 |

## 요구사항

### 페이지 헤더

- 제목: "캠페인 관리"
- 설명: "전체 캠페인 목록을 관리합니다"
- 신규 캠페인 생성 페이지로 이동하는 진입점

### 필터링/검색

사용자는 다음 조건으로 목록을 좁힐 수 있어야 한다.

| 필터 | 설명 |
|------|------|
| 키워드 검색 | 캠페인명으로 검색 |
| 채널 | Google Ads, Meta Ads, Naver SA, Kakao Moment, X Ads 중 선택 |
| 기간 | 캠페인 운영 기간 범위 |
| 상태 | 전체, 활성, 일시정지, 종료 |

필터는 동시에 여러 개를 적용할 수 있다.

### 캠페인 목록

"전체 캠페인" 제목, "총 50개 캠페인" 부제. 내보내기 기능 제공.

표시 항목:

| 항목 | 설명 | 정렬 |
|------|------|------|
| 선택 | 전체선택/개별선택 | - |
| 캠페인 ID | 시스템 식별자 | 가능 |
| 캠페인명 | 캠페인 이름 | 가능 |
| 상태 | 활성 / 일시정지 / 종료 | 가능 |
| 채널 | 집행 매체 | 가능 |
| 활성 토글 | 캠페인 활성/비활성 즉시 전환 | - |
| 시작일 | 캠페인 시작일 | 가능 |
| 종료일 | 캠페인 종료일 | 가능 |
| 노출수 | 총 노출 횟수 | 가능 |
| 클릭수 | 총 클릭 횟수 | 가능 |
| CTR | 클릭률 | 가능 |
| 지출 | 총 집행 금액 (원화) | 가능 |
| 액션 | 수정, 복제, 삭제(삭제는 위험 액션) | - |

### 사용자 액션

- 행 클릭: 캠페인 상세 페이지(`/campaigns/{id}`)로 이동
- 활성 토글: 캠페인을 활성 ↔ 비활성으로 즉시 전환
- 선택된 행 추적 가능

### 데이터 요건

- 한 페이지 20행, 총 50행 목 데이터
- 다양한 상태와 채널을 골고루 혼합 (페이지네이션 테스트 가능)

## 기술 제약

- 한국어 UI, 원화 표기
- 단일 페이지 컴포넌트, 목 데이터 인라인
